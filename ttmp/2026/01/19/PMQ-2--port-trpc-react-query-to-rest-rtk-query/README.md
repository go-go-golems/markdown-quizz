## Structure

- **design/**: Design documents and architecture notes
- **reference/**: Reference documentation and API contracts
- **playbooks/**: Operational playbooks and procedures

## Getting Started

Use docmgr commands to manage this workspace:

- Add documents: `docmgr doc add --ticket PMQ-2 --doc-type design-doc --title "My Design"`
- Import sources: `docmgr import file --ticket PMQ-2 --file /path/to/doc.md`
- Update metadata: `docmgr meta update --ticket PMQ-2 --field Status --value review`
